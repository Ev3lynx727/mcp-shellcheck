# ShellCheck MCP Server v0.1.0

A Model Context Protocol (MCP) server that provides shell script linting via ShellCheck.

## Features
- File and inline script analysis
- Support for bash, sh, dash, ksh, ash
- Configurable checks and severity filtering
- Works with OpenCode, Claude, Cursor

## Installation
```bash
uvx --from https://github.com/Ev3lynx727/mcp-shellcheck/releases/latest shellcheck-mcp-server
```

## Usage
Configure in your MCP client or use directly with OpenCode.
